# 考编备考全攻略生成器
## 产品文档 v2.0 · 增强版

> 考生填写基本信息，系统自动生成专属岗位分析、城市策略、月度计划、周课表、行测/申论/公基专项计划、考试节点倒排和完整备考报告。

---

## 一、产品概述

考编备考全攻略生成器是面向公务员考试、事业单位考试及相关考试备考考生的智能规划工具。产品核心价值在于：**考生只需填写基本信息，即可一键获得一份完整的、个人化的备考全攻略。**

### 设计目标

- 降低备考规划门槛，避免考生在"怎么学"上走弯路
- 根据专业、身份、城市、考试类型自动匹配最优策略
- 提供可直接执行的月度计划、周课表和专项训练方案
- 让备考节奏清晰可视，减少焦虑、提高上岸率

## 七、学习规划逻辑

### 7.1 四阶段划分

| 阶段 | 时间比例 | 核心任务 | 强度 |
|------|----------|----------|------|
| 第一轮·夯实基础 | 前 35% | 系统学习行测五大模块，申论建立归纳概括/综合分析/对策框架 | 每日6-8h |
| 第二轮·强化提升 | 35%–60% | 专项突破弱项，大量真题训练，建立错题本和限时节奏 | 每日8-10h |
| 第三轮·冲刺模考 | 60%–82% | 全真套卷、申论限时、时政热点、岗位表研究、查漏补缺 | 每日10-12h |
| 第四轮·面试毕业 | 后 18% | 结构化面试训练，体检政审材料，毕业论文同步推进 | 每日4-6h |

### 7.2 学习强度分类

| 每日时间 | 强度类型 | 课表策略 |
|----------|----------|----------|
| 3–5小时 | 压缩型 | 以真题复盘和核心题型为主，不追求刷题量 |
| 6–8小时 | 标准型 | 按七天标准课表执行，每周1套行测+1套申论 |
| 9–12小时 | 冲刺型 | 每周2套行测+2套申论，加入公基专项训练 |

## 八、行测五大模块专项计划

| 模块 | 目标正确率 | 重点知识点 | 训练建议 |
|------|------------|------------|----------|
| 资料分析 | 90% | 增长率、比重、倍数、速算技巧 | 每天20题限时25分钟，争取满分区 |
| 判断推理 | 85% | 图形/定义/类比/逻辑判断 | 建立题型模板，理科生重点冲高分 |
| 言语理解 | 80% | 逻辑填空、片段阅读、语句表达 | 每天30-40题，积累高频成语和标志词 |
| 数量关系 | 70% | 工程/行程/排列组合/最值 | 后置策略，挑会做的先做，不恋战 |
| 常识判断 | 60% | 时政、法律、科技、经济、人文 | 碎片化积累，重点时政和法律高频点 |

### 考场做题顺序（推荐）

```
常识判断(10min) → 资料分析(25min) → 判断推理(35min) → 言语理解(35min) → 数量关系(15min)
```

## 九、申论六大题型专项计划

| 题型 | 核心方法 | 训练频率 |
|------|----------|----------|
| 归纳概括 | 要点化、序号化，用材料原词，问什么答什么 | 每周2次以上 |
| 综合分析 | 解释→原因→影响→对策→升华 | 每周1次 |
| 提出对策 | 问题导向，主体明确，措施具体可执行 | 每周1次 |
| 贯彻执行 | 背格式（讲话稿/简报/倡议书/调研报告） | 每周1次 |
| 大作文 | 五段三分式，积累标题和分论点素材 | 每2周1篇+批改 |
| 卷面表达 | 每天练字10-20分钟，控制字距行距 | 每天 |

> **申论学习核心原则：** 材料阅读和复盘的效果远高于死背模板。从备考第一周起即需有计划练习，不要等到冲刺再学。

## 十二、面试专项规划

### 四大题型

| 题型 | 答题结构 | 核心要点 |
|------|----------|----------|
| 综合分析 | 表态→原因→影响→对策→升华 | 多角度思考，避免空话 |
| 计划组织 | 目的→对象→方式→内容→保障→评估 | 逻辑清晰，步骤完整 |
| 人际沟通 | 反思→理解→沟通→推动解决 | 先己后人，主动担当 |
| 应急应变 | 稳局面→分轻重→逐一解决→复盘预防 | 不慌乱，有条理 |

### 面试高分技巧

- 每天对镜子/录音练3道题，每周2次全真模拟
- 着装正装，深色为主，简洁大方
- 眼神交流，看考官，控制语速
- 结合报考岗位的具体工作内容回答
- 每道题控制在3-4分钟内

## 十三、重要注意事项

1. **应届身份保护** — 不要随便签劳动合同或三方协议，避免失去应届生资格
2. **报名策略** — 每次报名配置冲刺+稳定+保底岗位，不要全部押注同一城市
3. **错题管理** — 错题本必须有，三遍法（当天→3天→7天）复盘，做错10000题不如搞透1000道错题
4. **申论不能忽视** — 申论占总分50%权重，从备考第一天起就必须有计划练习
5. **面试早准备** — 笔试结束后立即进入结构化面试准备，不要等成绩出来再开始
6. **选调资格核验** — 党员条件、学生干部条件、学校名单需要提前核验，缺一不可
7. **信息渠道** — 国家公务员局（bm.scs.gov.cn）、省人事考试网、学校就业办是最权威的报名信息来源

## 十四、技术说明（HTML 原型）

### 文件特性

- **独立运行**：单文件，无外部依赖，直接在浏览器打开即可使用
- **实时响应**：所有输入字段改变时立即触发右侧所有内容更新
- **打印支持**：内置打印功能，浏览器打印/另存为PDF
- **复制导出**：报告页面支持一键复制完整文本到剪贴板
- **响应式布局**：桌面两栏、移动端自动单栏

### 专业匹配算法

```
输入专业名称
  → 遍历专业库（5个分类，按优先级从上到下）
    → 对每个分类的关键词数组做 indexOf 匹配
      → 命中第一个分类 → 返回对应岗位列表
  → 所有分类未命中 → 归入"通用综合类"（兜底）
```

### 备考周期计算

```
总月数 = (毕业年 - 起点年) × 12 + (毕业月 - 起点月) + 1
最小值 = 1 个月
四阶段占比:
  基础阶段  前 35%
  强化阶段  35% ~ 60%
  冲刺阶段  60% ~ 82%
  面试阶段  后 18%
```

### 倒排天数算法

```
剩余天数 = Math.ceil((考试日期 - TODAY) / 86400000)
状态判断:
  剩余 > 120天  → 绿色 OK
  60 < 剩余 ≤ 120天 → 橙色 WARN
  0 < 剩余 ≤ 60天  → 红色 BAD
  剩余 ≤ 0     → 灰色（已过期）
```
