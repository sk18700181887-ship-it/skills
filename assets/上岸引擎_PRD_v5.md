# 上岸引擎 · PRD 产品需求文档 v5

## 一、产品定位

**上岸引擎（Anan Engine）** —— AI 智能公考备考 SaaS 平台，覆盖从"了解考公"到"面试上岸"的完整 7 阶段路径，每个环节嵌入 AI 能力，形成免费引流 → 单点付费 → VIP 订阅 → 协议班高客单的商业闭环。

- 品牌名：上岸引擎
- 英文标识：ANAN ENGINE
- Slogan：AI 智能陪练，稳步上岸
- 目标用户：公务员 / 事业单位 / 选调生考生
- 设计风格：Topology.vc 极简暗色（纯黑基底 + Lime 信标 + 衬线标题 + 无圆角）

## 三、特色功能模块

### 3.1 开场 Hero 页（`/`）
- **电影级开场动画**：数字雨 + 扫描线 + 品牌名逐字打字机 + 数据指标飞入
- **品牌展示**：「上岸引擎」大号衬线体逐字显示 + 「ANAN ENGINE」英文标识
- **核心数据**：150 万考生 / 68% 上岸率 / 12,800 岗位 / 31 省份
- **入口按钮**：「进入系统」呼吸动画，点击后淡出过渡到学习中心

### 3.2 全国公考可视化（`/map`）
- **全国概览**：28 省份竞争比热力网格 + 难度筛选 + 省份详情卡片 + 排行榜
- **地市详情**：山东省 16 地市难度分布 + 竞争比柱状图
- **岗位分布**：9 类岗位占比环形图 + 横向条形图
- **国考系统**：8 大国考系统竞争排行 + 岗位数对比

### 3.3 情绪陪伴体系
- **每日心情选择**：5 种心情 + AI 即时鼓励语（Dashboard 内嵌）
- **上岸日记**（`/diary`）：心情记录 + 日记书写 + AI 陪伴回复 + 情绪树洞 + 能量站
- **AI 陪伴浮窗**：全局右下角可唤起聊天，6 个快捷回复 + 自由输入
- **连续打卡**：本周打卡进度 + 徽章墙（7 级里程碑）+ 距下一徽章倒计时

### 3.4 智能题库（`/practice`）
- 行测 5 大模块分类刷题
- 完整做题界面（选项选择 + 即时反馈 + 解析展开）
- 免费额度 + 错题本
- 五模块进度可视化

### 3.5 申论 AI 批改（`/shenlun`）
- 五维雷达图评分（立意/结构/论证/语言/卷面）
- 逐句点评 + 改进建议
- 历史批改记录
- 免费体验 → VIP 无限次

### 3.6 公基 AI 图谱（`/gongji`）
- 6 大模块知识树（政治/法律/经济/管理/公文/科技）
- AI 记忆助手 + 知识卡片
- 全模块覆盖率环形图
- 逐模块进度可视化

### 3.7 模考中心（`/mock`）
- 全国排期表 + 报名入口
- 成绩趋势折线图（8 个月数据）
- 上岸概率仪表盘（SVG 圆弧）
- 四维能力分析

### 3.8 AI 面试模拟（`/interview`）
- 4 大题型（综合分析/计划组织/人际关系/应急应变）
- 模拟面试流程（读题→作答→点评）
- 实时点评 + 高分技巧推荐
- ¥19.9/次 → VIP 无限

## 五、技术栈

- **框架**：Next.js 16 (App Router) + React 19 + TypeScript 5
- **UI**：shadcn/ui (Radix UI) + Tailwind CSS 4
- **设计**：Topology.vc 风格暗色主题（纯黑 #000 + Lime #b4ff39 + 衬线标题 + 无圆角）
- **动画**：CSS-only（数字雨、扫描线、打字机、发光脉动、渐变流动、进度条动画）
- **数据**：客户端 Mock（`src/lib/data.ts`），待接入后端
- **部署**：静态导出 (`output: 'export'`)，端口 `DEPLOY_RUN_PORT`

## 七、动态效果规格

| 效果 | 应用位置 | 实现方式 |
| --- | --- | --- |
| 数字雨 | 开场 Hero 背景 | 8 列随机数字下落 CSS 动画 |
| 扫描线 | 开场 Hero 全屏 | 水平光线循环扫描 |
| 打字机 | 品牌名逐字显示 | CSS steps() + opacity |
| 文字发光脉动 | 章节标题、数据标签 | glow-text keyframes + text-shadow |
| 渐变流动 | Dashboard 背景 | gradient-flow keyframes |
| 浮动光球 | 全局背景装饰 | 3 个半透明 lime 光球 float 动画 |
| 数字滚动 | KPI 数值 | countUp keyframes |
| 卡片弹入 | 卡片组件 | card-pop keyframes + stagger |
| 进度条填充 | 各类进度条 | progress-fill keyframes |
| 呼吸脉动 | CTA 按钮 | breathe keyframes |
| 鼠标光晕 | 全局跟随 | MouseGlow 组件 |
| Hover 高亮 | 卡片/链接 hover | 边框亮起 + 文字变 lime |
