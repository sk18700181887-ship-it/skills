# DESIGN.md · 上岸引擎（AI 智能公考备考平台 · 商业版）

## 产品灵魂 · 意象锚点
**"清晨的备考自习室 × 精英训练营"**：靠窗木质长桌上翻开的错题本、旁边一枚朱砂印章、书角一张烫金 VIP 会员卡。既有考编场景的克制与秩序，又有一个专业训练团队的商业质感。

严格回避的第一直觉：花哨科技蓝渐变、少年感紫粉赛博、廉价橙红对比。

## 品牌
- 中文品牌名：**上岸引擎**
- 英文标识：**Anan Engine**
- Slogan：**"AI 智能陪练，稳步上岸"**
- Logo 视觉：朱砂色印章 + "上岸"两字（`stamp` 效果）

## 视觉策略
- **档案纸质感 + 章印视觉**：档案纸米灰底 + 白色卡片 + 极细分隔线 + 朱砂点缀。信息层级依赖字号/字重/间距。
- **VIP 视觉资产**：深墨色渐变 + 烫金字（`.vip-gradient` / `.vip-text`），只用于会员权益、协议班、锁定内容。营造克制但珍贵的付费感。
- **多页面导航**：左侧固定 Sidebar（220px），主内容区自适应。移动端顶部横栏。

## Design Tokens

### 色彩（写在 `globals.css` :root）
- 主色 Primary：朱砂橘 `oklch(0.72 0.14 55)`（约 #F3A04C）
- 主色深：`#E58522` 用于 hover
- 前景 Foreground：接近墨黑 `oklch(0.22 0.01 60)`
- 背景 Background：档案纸米灰 `oklch(0.98 0.004 80)`
- 卡片 Card：纯白 `oklch(1 0 0)`
- 语义：`--ok`（墨绿 通过/正确）/ `--warn`（琥珀 注意）/ `--err`（朱红 错误/紧张）
- VIP：`--vip` 深墨 + 烫金渐变

### 字体
- 主体：`PingFang SC` / `Hiragino Sans GB` / `Microsoft YaHei` 系统栈（已在 `--font-sans`）
- 品牌/数字标题：可用 `--font-serif`（`Noto Serif SC`、`Songti SC`）体现"公文"气质，仅用于 Hero 主标题、KPI 数字
- 尺寸：正文 13-14 / 卡片标题 15 / 页面标题 20-22 / KPI 24-32

### 圆角 / 阴影 / 间距
- 圆角：控件 8px、卡片 12px、Chip 999px、大卡 16px
- 阴影：不用彩色发光；卡片仅 `border` + 极轻 `shadow-sm`
- 间距节奏：8 / 12 / 16 / 20 / 24（px）

### 动效
- 全站仅保留 hover/press `.15s ease` 过渡
- VIP 卡上"限时优惠"用 `.shimmer` 微光扫过（仅营销点）
- 无入场动画、无粒子

## 布局
- 桌面（≥ 1024px）：Sidebar 220px + 主内容 max-w-6xl，右侧留白
- 平板：Sidebar 折叠为图标 64px
- 移动端（< 768px）：Sidebar 变为顶部 Sheet 抽屉

## 组件规范
- **按钮**：主按钮朱砂橘、次按钮浅灰底、危险按钮墨红
- **VIP 徽章**：`.vip-gradient` 深墨底 + `.vip-text` 烫金字，圆角 8-999px
- **锁定内容**：内容半透明 + 顶层遮罩显示 🔒 + "开通 VIP 解锁"按钮
- **进度条**：轨道 `--muted`，填充朱砂橘线性渐变
- **状态 Chip**：`ok/warn/err` 三档，14% 半透明底 + 26% 边

## 内容 / 文案调性
- 简体中文，语气务实、专业、克制
- 商业术语真实：**免费额度、限时优惠、协议班退款、连续打卡**
- 数字均半角，中文标点全角
- 不出现"YYDS/上岸稳了/绝绝子"等口语

## 商业闭环视觉指引
1. **免费引流点**（左上角标 `免费`）：AI 匹配预览、每日 10 题、公开课
2. **付费单点**（右上角标价格 Chip）：申论批改 `¥9.9`、深度报告 `¥29.9`、人工批改 `¥199`
3. **VIP 转化 CTA**（页面右侧固定卡）：黄金视觉 + 限时倒计时
4. **协议班转化**（专属专题页）：全屏营销 + 上岸案例 + 退款承诺印章

## 设计禁忌
- 不使用彩色渐变按钮（除 VIP 深墨 + 烫金）
- 不使用玻璃拟态、Neumorphism、粒子背景
- 不用 Emoji 作为主要图形（Lucide icons 优先）
- 不加入场动画、滚动视差
