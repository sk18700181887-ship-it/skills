# 跨领域组合答题法

## R — 原文 (Reading)

> 部分规范词的使用场景横跨多个领域：如"拓宽引才渠道"同时出现在基层治理和人才队伍建设模块；"发展旅游业"在产业和文化模块均有涉及。实际申论答案往往需要跨领域组合多个规范词，按照"问题→对策→效果"的逻辑链组织，而非堆砌规范词。
>
> — 《申论规范词积累课程》，跨模块分析

---

## I — 方法论骨架 (Interpretation)

实际申论答案很少只涉及一个政策领域。一道关于"乡村振兴"的题目，可能同时需要产业（降本增效、销售渠道拓宽）、文化（乡风文明、文旅融合）、治理（干部下沉、群众自治）、人才（优化人才结构、拓宽引才渠道）四个领域的规范词。

组合的核心逻辑是"主领域骨架 + 次领域补充 + 逻辑链串联"：
1. **定主领域**：判断材料的核心议题属于8大领域中的哪一个，在该领域选取核心规范词作为答案骨架。
2. **补次领域**：扫描材料是否涉及其他领域的关键词或做法，在次要领域中选取补充规范词增加答案丰满度。
3. **串逻辑链**：规范词不能堆砌，必须按逻辑链排列。逻辑链有三种类型：
   - **问题→对策链**：先点明问题定性词，再给出对应对策词（如"同质化→推进文旅融合"）
   - **措施→效果链**：先列对策规范词，再接效果/特征规范词（如"统一生产标准→提升产品认可度"）
   - **时间序列链**：按材料事件发展的先后顺序排列（如"环保错位"中先开发后治理的时间线）

最终目标是从"选对词"跨越到"排好队"——让规范词之间形成有机的逻辑关系，而不是散落的关键词清单。

## A1 — 书中的应用 (Past Application)

### 案例 1: Q县红色旅游（文化+产业交叉）
- **问题**: 材料描述Q县利用红色英模资源发展旅游，同时带动农产品销售
- **方法论的使用**: 先确定主领域为文化（红色文化资源），选取"依托文化资源""推进文旅融合"作为骨架；次领域为产业，补充"发展旅游业""销售渠道拓宽"作为丰满度。逻辑链为"措施→效果"：围绕红色主题选路线（措施）→游客增多→苹果卖出几千斤（效果）
- **结论**: 答案需组合文化领域（依托文化资源、推进文旅融合）+ 产业领域（发展旅游业、销售渠道拓宽）的规范词
- **结果**: "推进文旅融合"精准概括了文化+产业的交叉本质

### 案例 2: G省"百校千企"平台（教育+产业交叉）
- **问题**: 材料描述G省搭建百校千企平台，组织技工院校与重点企业深度对接
- **方法论的使用**: 主领域为教育（技工院校、人才培养），选取"校企合作"作为骨架；次领域为产业（企业发展、产业需求），补充"人才共育、基地共建"。逻辑链为"措施→效果"：搭建平台（措施）→人才共育+基地共建（效果）
- **结论**: "校企合作"本身就是教育+产业的交叉规范词，同时涉及两个领域
- **结果**: 深化校企合作精准概括了产教融合的实践经验

### 案例 3: S村林下核桃经济（生态+产业交叉）
- **问题**: 材料描述S村荒山绿化后利用林下土地发展核桃种植，实现经济效益
- **方法论的使用**: 主领域为生态（荒山变绿、绿化），选取"促进林下经济发展"作为骨架；次领域为产业（核桃种植、赚钱），补充"降本增效"相关表达。逻辑链为"措施→效果"：荒山绿化（措施）→利用林地资源→村民增收（效果）
- **结论**: 答案需组合生态（绿化、保护性开发）+ 产业（林下经济、增收）的规范词
- **结果**: "促进林下经济发展"同时体现了生态保护与产业发展的双重收益

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. 申论材料明显涉及两个以上政策领域的交叉（如文旅融合、校企合作、林下经济等复合型话题）
2. 用户已经选好了若干规范词，但觉得答案"不够全面"或"遗漏了什么"
3. 用户需要组织答案的逻辑结构——将散落的规范词排成有条理的答案
4. 用户在答"概括经验""提出对策"类题目时，不确定答案应该覆盖哪些维度

### 语言信号 (用户的话里出现这些就应激活)

- "这个材料涉及好几个方面，怎么组织答案"
- "我的答案感觉不够全面，是不是漏了什么领域"
- "这些规范词之间怎么排列比较好"
- "乡村振兴/数字乡村类的题目，答案应该覆盖哪些维度"
- "帮我把这些规范词按逻辑串起来"

### 与相邻 skill 的区分

- 与 `sanmoshi-fanyi` 的区别: 三模式翻译解决的是"单段口语→单个规范词"的翻译问题；跨领域组合解决的是"多个规范词→完整答案"的组织问题。翻译是选词，组合是排队。
- 与 `wenti-celue-yinghe` 的区别: 问题-对策映射解决的是"问题→对策"的精准匹配（选哪个词）；跨领域组合解决的是"选好的词怎么排列"（怎么排队）。前者管精准，后者管完整。
- 与 `fen-weidu-moban`（分维度对策模板）的区别: 分维度对策提供的是单一领域内的维度框架（如人才的引-育-留）；跨领域组合提供的是跨领域之间的组合逻辑和逻辑链串联。

## E — 可执行步骤 (Execution)

当 skill 被激活后, agent 应按以下步骤执行:

1. **判断主领域和次领域**
   - 阅读全部材料，判断核心议题属于8大领域（教育、产业、治理、民生、生态、文化、执法、人才）中的哪一个
   - 标记材料中涉及的其他领域关键词
   - 完成标准: 明确标注"主领域=X，次领域=Y、Z"

2. **在主领域选取核心规范词（骨架）**
   - 调用 `wenti-celue-yinghe` 和 `xinhao-ci-fanyi` 在主要领域中选取3-5个最核心的规范词
   - 这些规范词构成答案的主体框架
   - 完成标准: 主领域规范词已选定，覆盖题目的核心要点

3. **在次领域选取补充规范词（丰满度）**
   - 扫描次要领域中是否有材料明确提及的做法或成效
   - 每个次领域选取1-3个补充规范词
   - 判停条件: 若次领域的规范词与主领域存在功能重叠（如同一个词出现在两个领域中），不重复选取

4. **确定逻辑链类型**
   - 根据题目问法选择逻辑链类型:
     - 题目问"存在问题" → 使用"问题定性"链
     - 题目问"对策/建议/做法" → 使用"问题→对策"或"措施→效果"链
     - 题目问"经验/启示" → 使用"措施→效果"链
     - 题目问"变化/过程" → 使用"时间序列"链
   - 完成标准: 明确选定一种逻辑链类型

5. **按逻辑链排列规范词**
   - 将选定的所有规范词按逻辑链顺序排列
   - 确保每个规范词在链中的位置合理（问题在前、对策居中、效果在后）
   - 检查相邻规范词之间是否存在因果或递进关系
   - 完成标准: 形成有序列表，每个词有明确的位置逻辑

6. **检查完整性与逻辑通顺**
   - 扫描是否有遗漏的政策领域（用8领域清单逐一核对）
   - 检查逻辑链是否通顺（问题→对策是否匹配、措施→效果是否因果成立）
   - 检查是否主次分明（主领域规范词占60%以上，次领域不超过40%）
   - 完成标准: 无遗漏领域、逻辑链无断裂、主次比例合理

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **单一领域的简单题目**: 如果材料只涉及一个政策领域且题目只要求列举对策，直接用 `sanmoshi-fanyi` 逐条翻译即可，不需要跨领域组合
- **纯翻译练习**: 用户在训练单个规范词的识别和翻译能力时，这是 `sanmoshi-fanyi` 的范畴

### 作者在书中警告的失败模式

- **堆砌规范词没有逻辑** (ce17): 答案列了很多规范词但之间没有逻辑关系，像散落的关键词清单而非有机整体。"只有砖块没有蓝图，盖不出房子"——规范词是砖块，逻辑链是蓝图。
- **跨领域但主次不分** (ce18): 材料明明以产业为主，考生却把文化、治理、人才的规范词都列上来，每个领域一两个，结果没有一个领域答透。主次不分导致答案面面俱到但蜻蜓点水。
- **忽视跨领域组合导致答案不完整** (ce18): 只在自己熟悉的单一领域内找规范词，忽略了材料的跨领域性质。例如材料讲"乡村振兴"涉及产业+文化+治理+人才，考生只写了产业相关规范词，遗漏其他维度。

### 作者的盲点 / 时代局限

- 课程主要聚焦"翻译"能力（口语→规范词），对"组合"能力的系统化训练不够充分。本 skill 是对这一盲点的补充——把组合逻辑显式化。
- 教材中的跨领域案例集中在文旅融合、校企合作等固定搭配，对于更复杂的三领域以上交叉（如数字乡村=治理+产业+民生+人才）覆盖较少，需要使用者自行推导。

### 容易混淆的邻近方法论

- 与 `sanmoshi-fanyi` 混淆: 翻译是"一词一译"，组合是"多词成链"。先翻译，再组合，顺序不可颠倒。
- 与 `fengweidu-duice` 混淆: 分维度对策是"同一领域内分几个角度"（如人才分引-育-留），跨领域组合是"不同领域之间怎么串联"。维度是纵向拆分，跨领域是横向拼接。
