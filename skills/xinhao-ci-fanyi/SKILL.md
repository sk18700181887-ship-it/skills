# 信号词→规范词触发链

## R — 原文 (Reading)

> 当资料中出现"村民自己的事务自己管理""职工代表大会""自治委员会"等类似表述时，通常可联想到"群众自治"。当资料中出现"专业合作社"等表述时，可概括为"发展集体经济"。资料中出现"积分排名""荣辱榜""红黑榜"等表述时，可概括为"完善奖惩机制"。
>
> — 《申论规范词积累课程》，全书规范词总结表·应用场景列

---

## I — 方法论骨架 (Interpretation)

这个 skill 的核心是建立**条件反射式的"信号词→规范词"触发对**，把翻译从"推理过程"变成"检索过程"。

它不是每次都从头分析"这段材料讲了什么→属于什么领域→应该用什么词"，而是在阅读材料时，一旦捕捉到特定关键词或典型表述，就直接触发对应规范词——就像看到红灯想到"停"一样自然。

运作机制分三层：
1. **信号识别**：材料中的特定关键词/短语/句式是"信号灯"，提示你应该联想到某个规范词
2. **触发匹配**：通过积累"信号词→规范词"映射表，实现一对一快速匹配
3. **领域辅助**：信号词天然携带领域信息（如"专业合作社"天然指向治理+产业），帮助快速定位

这个方法覆盖全部8大政策领域的关键信号词：教育（"教到哪算哪"→课程缺乏系统性）、产业（"给农作物上保险"→提供农业保险服务）、治理（"自治委员会"→群众自治）、民生（"政府买单"→购买公共服务）、生态（"先开发后治理"→环保错位）、文化（"大操大办"→移风易俗）、执法（"老百姓听得懂"→通俗性）、人才（"上级直接任命→打破"→优化人才选拔机制）。

本质上，它把 128 个规范词的使用条件浓缩成了一套"触发词典"——不是背词，而是背"什么时候用什么词"的条件。

## A1 — 书中的应用 (Past Application)

### 案例 1: "专业合作社"→发展集体经济

- **问题**: 真题材料描述"村民各过各的，现在有了专业合作社，把农产品销往全国，大家的腰包也越来越鼓"，需要快速识别核心规范词
- **方法论的使用**: 材料中出现"专业合作社"这一典型信号词，直接触发"发展集体经济"这一规范表达——不需要逐句分析"村民各过各的"代表分散经营，"合作社"代表组织化合作，"腰包鼓"代表经济效益提升，再从三个层面推导出结论
- **结论**: 信号词"专业合作社"直接命中规范词"发展集体经济"
- **结果**: 跳过推理环节，翻译速度从 2-3 分钟缩短到 10 秒

### 案例 2: "自治委员会"→群众自治

- **问题**: Q社区成立"居民议事协商会"，由业主代表组成，制定居民公约，需要概括这一做法
- **方法论的使用**: 材料中出现"自治委员会""居民议事协商会""群众自己的事务自己管理""职工代表大会"等任一表述，均为"群众自治"的触发信号
- **结论**: 信号词"居民议事协商会"→规范词"群众自治"
- **结果**: 避免了将其误译为"社区管理""民主监督"等近义但不精准的表述

### 案例 3: "积分排名/红黑榜"→完善奖惩机制

- **问题**: 六角塘村垃圾分类经验概括题，材料描述"排名头三位和末三位上榜，前五名奖1000元，末三位罚200元且取消福利"
- **方法论的使用**: 材料中"积分排名""荣辱榜""红黑榜"等表述是"完善奖惩机制"的固定触发词，无需分析具体奖惩金额和规则
- **结论**: 信号词"积分排名+红黑榜"→规范词"完善奖惩机制"（注意：必须同时包含"奖"和"惩"两方面）
- **结果**: 精准命中，避免因纠缠具体细节而遗漏核心规范词

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **做真题练习需要提速时** — 已经掌握三模式翻译的基本方法，但逐条推理太慢，需要建立快速通道
2. **问"看到XX材料应该用什么规范词"时** — 用户给出材料片段或关键词，需要快速给出对应的规范词
3. **需要快速判断材料属于哪个领域时** — 信号词天然携带领域信息，通过识别信号词可以秒判领域归属
4. **复习阶段检验信号词积累量时** — 用户希望快速过一遍"哪些信号词对应哪些规范词"

### 语言信号 (用户的话里出现这些就应激活)

- "看到这个词应该想到什么"
- "XX材料的关键词是什么"
- "快速翻译这段材料"
- "信号词有哪些"
- "这个表述对应哪个规范词"
- "帮我提速翻译"

### 与相邻 skill 的区分

- 与 `sanmoshi-fanyi` 的区别: sanmoshi-fanyi 是"分析方法"——教你怎么从头推理（先判断材料类型，再选翻译路径）；本 skill 是"快速通道"——跳过推理，直接用积累的信号词映射表匹配
- 与 `wenti-celue-yinghe` 的区别: wenti-celue-yinghe 解决"多个近义规范词选哪个"的精准度问题（定义-场景双重校验）；本 skill 解决"第一时间能不能想到这个词"的速度问题

## E — 可执行步骤 (Execution)

当 skill 被激活后，agent 应按以下步骤执行:

1. **扫描材料，标记关键词/典型表述**
   - 完成标准: 材料中的信号词已被标注出来（如"专业合作社""自治委员会""积分排名"等）
   - 判停条件: 已完成全文关键词扫描，或已找到 3 个以上有效信号词

2. **在信号词-规范词映射表中检索匹配**
   - 完成标准: 每个信号词都已尝试匹配对应的规范词
   - 辅助信息: 信号词通常来自8大领域的应用场景列，可通过领域关键词辅助定位
   - 判停条件: 所有信号词都已完成匹配或标记为"未命中"

3. **命中的信号词 → 直接使用该规范词**
   - 完成标准: 命中的信号词与规范词的配对已确认
   - 校验: 快速检查材料上下文是否与规范词的应用场景一致（防止机械匹配）

4. **未命中的信号词 → 回退到 sanmoshi-fanyi 的推理模式**
   - 完成标准: 未命中信号词已通过三模式翻译框架完成推理
   - 判停条件: 所有未命中项均已给出翻译结果

5. **新发现的信号对 → 加入映射表**
   - 完成标准: 新发现的"信号词→规范词"配对已记录，供后续使用
   - 判停条件: 无新增信号对，或新增项已记录

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **材料中没有明显信号词时** — 如果材料通篇是大段叙述、没有典型关键词，这个 skill 就失去了触发基础，应直接使用 sanmoshi-fanyi 从头推理
- **初次学习翻译方法时** — 先学会 sanmoshi-fanyi 的推理过程，再用本 skill 提速。没有理解推理逻辑就追求速度，等于"还没学会走就想跑"
- **需要区分近义规范词时** — 本 skill 解决的是"能不能想到"的速度问题，不解决"选得准不准"的精度问题；精度问题交给 wenti-celue-yinghe

### 作者在书中警告的失败模式

- **机械匹配（最常见陷阱）**: 看到"合作社"就写"发展集体经济"，但材料语境其实是讲"专业合作社提供农机租赁"——此时应译为"配备农机设备，提供租赁服务"而非"发展集体经济"。信号词是提示而非定论，必须结合上下文校验。
- **死记硬背但不识别触发场景**: 能默写 128 个规范词的释义，但看到新材料时无法激活对应信号词。背了"群众自治"的释义不等于看到"业主代表组成议事协商会"能立刻联想到它。
- **混淆硬件信号与软件信号**: 材料说"没有篮球场"→ 应选"完善文化设施"（硬件问题）；材料说"活动老是一套"→ 应选"创新文化活动形式"（软件问题）。如果把这两类信号词混在一起，翻译方向就完全错了。
- **信号词过度泛化**: 看到"向下"就笼统写"下沉"而不区分类型——"干部下沉"（人到基层去）、"政务服务下沉"（事权下放）、"党员下沉"（组织延伸）是三个完全不同的规范词。

### 作者的盲点 / 时代局限

- 信号词-规范词映射表依赖真题积累，对于未出现过的新表述，本 skill 无法覆盖
- 教材侧重"一对一"的信号词触发，但实际考试中同一段材料往往需要多个规范词组合——此时需要配合 kuailingyu-zuhe（跨领域组合答题法）

### 容易混淆的邻近方法论

- 与"政策领域快速诊断法"的区别: 领域诊断是判断"材料属于哪个领域"（8选1），信号词触发是判断"材料对应哪个规范词"（128选1），前者是后者的辅助步骤
- 与"同义表达归一法"的区别: 归一法是"多种不同表述指向同一个规范词"的识别能力，信号词触发是"看到特定关键词直接联想"的匹配速度，两者互补但侧重点不同
